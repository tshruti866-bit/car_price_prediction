# 🚗 Car Price Prediction using Linear Regression

## 📌 Project Overview
This project predicts the price of a car based on its mileage using a Linear Regression model built with Python and Scikit-learn.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 📊 Features
- Data visualization using scatter plot
- Data preprocessing
- Train-test split
- Linear Regression model
- Model evaluation using R² Score and MAE
- Car price prediction based on mileage

## 📈 Results
- **R² Score:** 0.94
- **MAE:** 52,500

## 📂 Project Files
- `car_price_prediction` – Jupyter Notebook containing the complete implementation

## 🚀 How to Run
1. Clone this repository.
2. Install the required libraries:
   ```
   pip install pandas numpy matplotlib scikit-learn
   ```
3. Open `car_price_prediction` in Jupyter Notebook or VS Code.
4. Run all cells to reproduce the results.
